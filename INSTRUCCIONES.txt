APP CONTROL DE PRODUCTOS — PASOS PARA PUBLICAR

1. Entra en GitHub y crea un repositorio nuevo, por ejemplo: control-productos.
2. Sube todos los archivos de esta carpeta, conservando la carpeta icons.
3. Abre Settings > Pages.
4. En “Build and deployment”, selecciona “Deploy from a branch”.
5. Selecciona la rama main y la carpeta /(root), y guarda.
6. GitHub mostrará una dirección HTTPS parecida a:
   https://TU-USUARIO.github.io/control-productos/

INSTALAR EN IPHONE
- Abre la dirección con Safari.
- Pulsa Compartir.
- Pulsa “Agregar a pantalla de inicio”.

INSTALAR EN ANDROID
- Abre la dirección con Chrome.
- Pulsa el menú de tres puntos.
- Pulsa “Instalar aplicación” o “Agregar a pantalla principal”.

IMPORTANTE
Los registros se guardan en el navegador del dispositivo. Usa “Exportar CSV” periódicamente como copia de seguridad.


Esta versión usa como icono el logo de Palm Springs Bakery proporcionado por el usuario.

VERSIÓN V7:
- Se reorganizó la interfaz en secciones: Inicio, Semana, Mes y Año.
- Inicio mantiene la entrada diaria, edición y resumen diario/semanal.
- Semana imprime recibo de 58 mm agrupado por días.
- Mes imprime reporte A4 con resumen, detalle de productos y totales.
- Año imprime reporte A4 con resumen anual por meses.
Para actualizar en GitHub Pages, sube todos estos archivos y confirma con Commit changes.
